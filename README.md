### Dependencies 

- Python3
- Django (pip-module)

    ``` bash
    python3-pip install django
    ```

### Local Setup Web Server

``` bash
git clone https://github.com/thekarananand/taskcontrol.git

cd ./taskcontrol

python3 manage.py makemigration
python3 manage.py migrate
```

### Run Web Server

``` bash
python3 manage.py runserver
```
